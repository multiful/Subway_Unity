## 프로젝트 개요

- 장르: 2D 비주얼 노벨, 아케이드
- 한 줄 소개: 지하철을 배경으로 한 선택지 기반 스토리 게임으로, 플레이어의 선택에 따라 엔딩이 달라지는 구조
- 타깃 플랫폼: Android / iOS
- 팀 구성: 기획 2 / 프로그래머 3 / 디자이너 1

## 주요 기능

- 선택지에 따른 스토리 분기
- 대사 스킵 / 자동 진행 / 로그 보기 UI
- 스코어/호감도 시스템 및 결과 화면
- 세이브/로드, 옵션(사운드, 텍스트 속도) 기능

## 내 역할

- Unity를 이용한 '빠르게환승' 미니게임 전체 설계 및 구현
- 담당 스토리 구간의 UI 레이아웃 및 C# 스크립트 작성
- 선택지·호감도·엔딩 분기 로직 설계 및 구현
- 세이브/로드 UI 및 직렬화 로직 구현

## 사용 기술
- Game Engine: Unity
- Language: C#
- Version Control: Git / GitHub
