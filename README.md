## 프로젝트 개요

- 장르: 2D 비주얼 노벨, 아케이드
- 한 줄 소개: 지하철을 배경으로 한 선택지 기반 스토리 게임으로, 플레이어의 선택에 따라 엔딩이 달라지는 구조
- 타깃 플랫폼: Android / iOS
- 팀 구성: 기획 2 / 프로그래머 3 / 디자이너 1

## 주요 기능

- 선택지에 따른 스토리 분기
- 대사 스킵 / 자동 진행 / 로그 보기 UI
- 스코어/호감도 시스템 및 결과 화면
- 세이브/로드, 옵션(사운드, 텍스트 속도) 기능

## 내 역할

- Unity '빠르게환승' 미니게임 구현
- 담당 대사 파트 UI, 스크립트 작성
- 선택지/호감도/엔딩 분기 로직 C#으로 구현
- 세이브, 로드 UI 및 기능 구현

## 사용 기술
- Unity
- 프로그래밍언어 : C#
- 버전관리 : Git / GitHub
